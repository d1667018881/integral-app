# 🚀 快速开始

## 📱 在 AndroidIDE 中 5 分钟完成

### 1️⃣ 导入项目

1. 打开 AndroidIDE
2. 点击 **Open Project**
3. 选择 `jifen-assistant` 文件夹

### 2️⃣ 等待 Gradle 同步

- 首次打开会自动下载依赖（约 2-3 分钟）
- 看到顶部进度条消失说明同步完成

### 3️⃣ 编译 APK

```
顶部菜单 → Build → Build APK
```

- 等待编译（首次约 5-10 分钟）
- 完成后会弹出通知

### 4️⃣ 找到 APK

```
app/build/outputs/apk/debug/app-debug.apk
```

- 长按 APK 文件 → 分享/保存到手机
- 或点击通知中的 **Locate** 按钮

### 5️⃣ 安装使用

1. 将 APK 传到手机
2. 允许"安装未知应用"
3. 点击 APK 安装
4. 打开"积分助手"使用！

---

## 🎯 使用示例

### 第一次使用

1. 打开应用
2. 输入工号：`30020116`
3. 输入目标积分：`100`
4. 点击 **▶ 开始执行**
5. 等待自动完成

### 修改设置

1. 点击 **⚙ 设置**
2. 修改接口 URL 或执行参数
3. 点击 **💾 保存设置**
4. 返回主页生效

## ⚠️ 常见问题

### Gradle 同步失败

```
解决：检查网络连接，或手动下载 Gradle
```

### 编译失败

```
解决：清理项目 → Build → Clean Project
```

### APK 安装失败

```
解决：允许安装未知应用 → 设置 → 安全 → 未知来源
```

### 中文显示异常

```
不可能！原生 Android 完美支持中文 ✅
```
